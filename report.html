1. Project Overview

This project involved building a fully responsive personal portfolio website with separate pages for About, Projects, and Contact. The site includes smooth animations, mobile navigation, and a working dark mode.

2. Steps to Build

  • Designed a responsive landing page using HTML, CSS and JS.
  • Created separate HTML files: index.html, about.html, project.html, report.html and contact.html.
  • Added animations and image shapes using CSS.
  • Implemented a mobile-friendly menu too with the help of JavaScript.
  • Enabled dark mode using a toggle button with the help of Javascript to make the website more user-friendly.
